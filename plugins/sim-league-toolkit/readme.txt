== Description ==

SIM League Toolkit is a plugin that provides blocks and admin pages to support building a website for motor racing league using any simulator or game

It allows you to use almost any theme and configure it to use the blocks provided to build your racing league web site, supporting a single simulator or multiple simulators

It allows you to create a collection of events as a championship or one off standalone events.  The results of an event can be entered manually or result files from sims can be uploaded an imported.
Either way championship standings are calculated and trophies can be awarded.

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Sim League Toolkit**.
3. Install and activate the Sim League Toolkit plugin.

= Manual installation =

1. Upload the entire `sim-league-toolkit` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Sim League Toolkit plugin.
