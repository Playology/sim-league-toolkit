== Description ==

SIM League Toolkit Theme is a theme that is built using the SIM League Toolkit plugin and provides a good starting point for building your own league website


== Installation ==

= Installation from within WordPress =

1. Visit **Themes > Add New**.
2. Search for **SIM League Toolkit Theme**.
3. Install and activate the SIM League Toolkit Theme theme.

= Manual installation =

1. Upload the entire `sim-league-toolkit-theme` folder to the `/wp-content/themes/` directory.
2. Visit **Appearance/Themes**.
3. Activate the SIM League Toolkit Theme theme.
